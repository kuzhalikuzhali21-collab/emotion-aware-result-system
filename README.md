# Emotion-Aware Result System using DeepFace

## 📌 Description
This project detects student emotions through facial expressions using AI and sends alerts to parents via email.

## 🧠 Technologies Used
- Python
- OpenCV
- DeepFace
- TensorFlow
- Keras
- SMTP (Email alerts)

## ⚙️ Features
- Detects emotions from facial expressions
- Uses deep learning for analysis
- Sends automatic email alerts to parents

## 🚀 Applications
- Smart education systems
- Student monitoring solutions
